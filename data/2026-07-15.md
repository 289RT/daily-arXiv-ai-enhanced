### [1] [AlphaFunctor: Bridging The Gap Between Protein Function Annotation and Property Prediction](https://arxiv.org/abs/2607.12196)
*Xiang Liu,Anna E. Yee,Josh V. Vermaas,Daniel R. Woldring,Guo-Wei Wei*

Main category: q-bio.BM

TL;DR: AlphaFunctor是一个基于范畴论的平台，直接通过蛋白质序列预测功能，并映射到下游性质预测，性能优于现有方法。


<details>
  <summary>Details</summary>
Motivation: 解决蛋白质功能注释和性质预测之间的巨大鸿沟，探索直接从序列预测功能与性质的可行性。

Method: 结合范畴论、拓扑谱理论、路径复合神经网络和蛋白质域分析，从序列预测功能（GO术语）并映射到性质预测。

Result: 在三个基准数据集上达到最先进的性能，且在未重新设计网络的情况下，优于特定任务的预测器。

Conclusion: AlphaFunctor为蛋白质功能与性质预测提供了统一框架，验证了直接从序列预测功能与性质的假设。

Abstract: The fundamental relationship among protein sequence, structure, function, and physicochemical properties is a central principle in biology. While in principle protein function and properties should be able to be derived directly from protein sequence, in practice protein function and property prediction methods have been designed around specific datasets and specific property or function subsets, leading to an enormous gap between function annotation and property prediction. To address these challenges, we introduce AlphaFunctor, a category theory based foundation model-like platform to bridge the gap between protein function annotation and property prediction. Based on the hypothesis that protein function and properties can be directly derived from protein sequence, AlphaFunctor predicts protein functions as represented by Gene Ontology terms directly from sequence. Using these function predictions, AlphaFunctor further maps protein functions using topological spectral theory, path-complex neural networks, and protein domain analysis onto downstream property prediction. AlphaFunctor is (pre)trained in nearly 0.6 million protein function data points to deliver the state-of-the-art protein function annotation on three benchmark datasets. Without task-specific network redesign, AlphaFunctor maps qualitative protein function annotation to various qualitative and quantitative protein property predictions, outperforming other dataset-specific and task-specific competing predictors.
